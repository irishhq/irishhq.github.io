  3. css语法规范

    1. css语法规范总结
      • 样式表: 选择器+样式声明(属性+值…)
      • 内联: 样式声明

    2. css样式表特征

      • 继承和层叠 : css层叠管理: 同时也是处理css的继承问题的关键

        • inherit关键字使得元素获取其父元素的计算值()注意隔代是否能继承问题,) 它可以应用于任何CSS属性，包括CSS简写“all”。
        • initial是将属性的初始值赋给元素 . initial 适用于所有的css 属性，包括css 简写属性(全局属性)[all]
          • 当元素的一个继承属性没有指定值时，则取父元素的同属性的计算值，只有文档根元素取该属性的概术中给定的初始值
          • 当元素的一个非继承属性没有指定值时，则取属性的初始值
        • unset如果为继承属性,相当于inherit;如果为非继承属性,相当于initial.
        • revert以前被称为default表示没有使用任何属性值.目前(2018-09)仅safari支持,尴尬…
        • all属性:包含三个值initial/inherit/unset

        ps: 

        - [CSS  Reference](https://www.w3schools.com/cssref/): 属性->Inherited
- 浏览器中元素调用样式的过程:

  7. 浮动定位

    1. 浮动概述
      浮动元素只会在当前行浮动,浮动元素会停靠在父元素的左边或右边或者平级元素左右.能让多个块级元素在一行内显示
    2. 浮动定位
      • 换行
      • 宽度自适应
      • 变为块级元素
      • 浮动元素高度不同,”卡主”现象
    3. float属性
    4. clear属性
      定义了元素的哪边上不允许出现浮动元素,

    5. float与overflow

      由于浮动元素会脱离文档流,所以不占用页面空间,所以会对父级元素高度带来一定的影响.如果父元素高度为0,对父元素产生高度坍塌
      解决方案:

      • 设置父元素浮动
      • 为父元素设置overflow hidden auto
      • 父元素中,追加子元素,并设置clear:both

    定位

  5. 定位概述
    1. 什么是定位
    2. 定位属性
      position
      将元素的position属性定义为relative/absolute/fixed,这个元素就叫已定位元素
已定位元素,才能用偏移属性top/bottom/left/right
  6. 定位方式
    1. 相对定位
      元素脱离文档流,但是原来的位置不被其他文档中元素占用
      自身元素微调
    2. 绝对定位
      • 元素脱离默认文档流,原来位置被其他没有脱离文档流的元素占用.
      • 所有的层叠效果,都要使用absolute
      • 绝对定位的参照物:
        • 如果该元素的父元素是已定位元素,以父元素的左上角为基点定位
        • 如果该元素的父元素不是已定位元素,向上再找一层,直到找到是定位元素的父级,以那层父元素为基准定位
        • 该元素没有是定位元素的祖先元素,以body为基准
    3. 堆叠顺序
    4. 固定定位

弹性布局

弹性布局是一种布局方式,为框模型提供最大的灵活性.
把父元素设置成弹性布局display: flex;
子元素的排列方向为主轴
交叉轴,与主轴垂直相交的
父元素—-容器
子元素—-项目
注意:

  • 设置为flex布局后,子元素的float,clear和verticall-align属性将失效
  • 项目允许修改尺寸
  • 容器的对齐方式text-align失效
    弹性布局中,子元素叫项目
  1. 容器属性:
    1. flex-direction
    2. flex-wrap
    3. flex-flow
    4. justify-content
    5. align-items
    6. align-content

    2. flex-grow

      转换

      transform

  • transform-function

    • 位移translate(x [,y]);
    • 缩放scale()

    • 旋转rotate():正值是顺时针连同X,Y轴旋转了,注意以下两句的区别

    • skew() 

      skewX(x)正值是逆时针,让元素向着X轴发生倾斜效果,X轴角度不变
skewY(y)正值是顺时针,让元素向着Y轴发生倾斜效果,Y轴角度不变

  • transform-origin:转换原点,默认center,center

    3D

  • 透视距离
    模拟人的眼睛到3D转换元素之间的距离
    视距越小,人的眼睛离3D元素越近,3D效果越强烈.
    为元素定义perspective,其子元素会获得透视效果
  • rotateX()以X轴为中心,旋转元素的角度(烤肉架)
  • rotateY()以Y轴为中心,旋转元素的角度(旋转门)
  • rotateZ()以Z轴为中心
  • rotate3D(x, y, z, deg);x,y,z大于0的数字表示参与旋转,取值为0,表示不参与旋转

    过渡

  1. 过渡概述
    1. 什么是过渡
      四要素:
      • 过渡属性
      • 过渡时间
      • 过渡函数:过渡的速度,方式等
      • 过渡延迟
    2. 触发过渡
      • 由用户的行为触发(点击,悬浮等)
      • 由元素的状态触发(hover,active,focus…)
      • 由JavaScript代码触发
  2. 过渡子属性
    1. 过渡属性transition-property
      all:一切能用过渡的属性,一律用过渡来显示
      可以设置过渡的属性
      1. 颜色
      2. 取值为数字的属性
      3. 转换效果
      4. 阴影效果
      5. 渐变效果
      6. visibility
    2. 过渡时间transition-duration( s | ms )
      默认0
    3. 过渡函数transiton-timing-function
      取值为预定义函数或贝塞尔曲线
      预定义函数
      • ease: 默认,慢速开始,快速变快,然后慢速结束
      • linear:开始至结束相同速度
      • ease-in: 以慢速开始,加速效果
      • ease-out:以慢速结束,减速效果
      • ease-in-out: 以慢速开始和结束,先加速再减速
    4. 过渡延迟transition-delay
    5. 简写属性transition
      transition: property duration timing-function delay
    6. 多个过渡
      多个值之间用逗号隔开
      transition-delay: 1s, 4s;
      过渡属性放在元素的声明的样式中,过渡”有去有回”;
      过渡元素放在触发属性(hover)中,只管去不管回
